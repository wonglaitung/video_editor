# 视频剪辑

从一段视频中剪辑出多个片段，然后合并成一个完整的视频文件。

## 用法

```
/video-clip <输入视频> -c <时间段1> -c <时间段2> ... [-o <输出路径>]
```

## 参数说明

- `<输入视频>`: 源视频文件路径（必需）
- `-c <时间段>`: 剪辑时间段，格式 `开始-结束`（可多次指定，必需）
- `-o <输出路径>`: 输出文件路径（可选，默认 `clipped_<原文件名>`）

## 时间格式

支持多种时间格式：
- 秒数：`30`, `120`
- 分:秒：`1:30`, `2:45`
- 时:分:秒：`0:01:30`, `1:02:30`

## 执行步骤

当用户调用此技能时，执行以下操作：

1. 解析用户提供的参数

2. 运行剪辑脚本：
   ```bash
   python3 video_clipper.py "<输入视频>" -c <时间段1> -c <时间段2> ... [-o "<输出路径>"]
   ```

3. 报告剪辑结果，包括：
   - 各片段的时间范围
   - 输出文件路径
   - 输出视频时长

## 示例调用

用户输入：
```
/video-clip video.mp4 -c 0:10-0:30 -c 1:00-1:30
```

执行：
```bash
python3 video_clipper.py "video.mp4" -c 0:10-0:30 -c 1:00-1:30
```

用户输入：
```
/video-clip input.mp4 -c 10-30 -c 60-90 -o highlight.mp4
```

执行：
```bash
python3 video_clipper.py "input.mp4" -c 10-30 -c 60-90 -o "highlight.mp4"
```

用户输入：
```
/video-clip movie.mp4 -c 0:01:00-0:02:30 -c 0:05:00-0:07:00
```

执行：
```bash
python3 video_clipper.py "movie.mp4" -c 0:01:00-0:02:30 -c 0:05:00-0:07:00
```

## 依赖

需要安装 moviepy：
```bash
pip install moviepy
```

## 功能特性

- 支持多种时间格式输入
- 自动验证时间段是否在视频时长内
- 保持原视频编码格式输出
- 支持多个片段按顺序合并
